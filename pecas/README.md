# ♟ Xadrez - Fundamentos e Técnicas Avançadas

Projeto desenvolvido para a disciplina de Programação.

## 📌 Objetivo

Aplicar conceitos de Programação Orientada a Objetos no desenvolvimento
da lógica de um jogo de xadrez, utilizando fundamentos e técnicas avançadas.

---

## 🧠 Conceitos Aplicados

- Programação Orientada a Objetos (POO)
- Abstração
- Encapsulamento
- Herança
- Polimorfismo
- Modularização
- Tratamento de Exceções
- Organização em Pacotes
